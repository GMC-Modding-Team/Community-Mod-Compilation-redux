# Goats of the Apocalypse (CDDA Mod)

**Version:** 0.1.0  
**Depends on:** `dda`

## What it does
- Adds a new wilderness creature: **goat** (`g`). It’s mostly shy but can butt you if provoked.
- Spawns in forests and fields, usually alone or in small groups.
- Adds a tiny craftable item: **goat bell** (crafting → Other), to confirm the mod is active.

## Install
1. Unzip `GoatsOfTheApocalypse-0.1.0.zip` into your `Cataclysm-DDA/data/mods/` folder.
2. Enable **Goats of the Apocalypse** when creating a world.
3. (Optional) Craft a *goat bell* from 2× scrap to verify.

## Notes
- Goats butcher like medium mammals using the base game's harvest data.
- Balance is intentionally light-touch; tweak spawn frequencies in `monster_groups/groups.json` to taste.
